## Instructions:


#### Start consul ­ЪќЦ№ИЈ
рЃњрЃљрЃЋрЃБрЃерЃЋрЃљрЃЌ `рЃЎрЃЮрЃюрЃАрЃБрЃџрЃўрЃА рЃљрЃњрЃћрЃюрЃбрЃў` рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃб рЃЏрЃЮрЃБрЃЊрЃерЃў:

```bash
npm run start:consul
```

#### Start:apps ­Ъџђ

рЃњрЃљрЃЋрЃБрЃерЃЋрЃљрЃЌ рЃЊрЃљ рЃЊрЃљрЃЋрЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃЮрЃЌ 2 рЃўрЃюрЃАрЃбрЃљрЃюрЃАрЃў `api-service`-рЃўрЃА рЃЊрЃљ 1 рЃўрЃюрЃАрЃбрЃљрЃюрЃАрЃў `webapp-service`-рЃўрЃА:

```bash
npm run start:apps
```


#### Start:loadBalancer

рЃЊрЃљрЃЋрЃАрЃбрЃљрЃарЃбрЃЮрЃЌ рЃџрЃЮрЃљрЃЊ-рЃЉрЃљрЃџрЃљрЃюрЃАрЃћрЃарЃў, рЃбрЃарЃћрЃцрЃўрЃЎрЃўрЃА рЃАрЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃЉрЃџрЃљрЃЊ:

```bash
npm run start:loadBalancer
```

#### рЃњрЃљрЃЋрЃБрЃерЃЋрЃљрЃЌ рЃарЃћрЃЦрЃЋрЃћрЃАрЃбрЃў `api`-рЃќрЃћ 2рЃ»рЃћрЃа, рЃЊрЃљрЃЋрЃўрЃюрЃљрЃ«рЃљрЃЋрЃЌ рЃарЃЮрЃЏ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃўрЃюрЃАрЃбрЃљрЃюрЃќрЃћ рЃЏрЃўрЃЋрЃљ рЃњрЃљрЃерЃЋрЃћрЃЉрЃБрЃџрЃў рЃарЃћрЃЦрЃЋрЃћрЃАрЃбрЃў.
```bash
  curl localhost:8080/api
```

#### рЃњрЃљрЃЋрЃБрЃерЃЋрЃљрЃЌ рЃАрЃбрЃарЃћрЃАрЃА рЃбрЃћрЃАрЃбрЃў `autocanonn`-рЃўрЃА рЃАрЃљрЃерЃБрЃћрЃљрЃџрЃћрЃЉрЃўрЃЌ, рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃњрЃљрЃЋрЃўрЃњрЃЮрЃЌ рЃарЃљрЃЏрЃЊрЃћрЃюрЃў рЃарЃћрЃЦрЃЋрЃћрЃАрЃбрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ рЃерЃћрЃБрЃФрЃџрЃўрЃљ 200 рЃЦрЃЮрЃюрЃћрЃЦрЃерЃћрЃюрЃќрЃћ 20 рЃгрЃљрЃЏрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў рЃЕрЃЋрЃћрЃюрЃА рЃЉрЃљрЃџрЃљрЃюрЃАрЃћрЃарЃА.

```bash
autocannon -c 200 -d 20 http://localhost:8080
```
